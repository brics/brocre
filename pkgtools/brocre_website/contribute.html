Contribute to BROCRE

If you want to contribute a software package to BROCRE please send an email to the BRSU group (jan.paulus / azamat.shakhimardanov / nico.hochgeschwender At h-brs.de) with the following information.

  • The name of the software package
  • Where to find the software
  • Which commit, tag or branch should be used
  • The categories which describes the software
  • The dependencies to other BROCRE packages
  • The version of the software *
  • The maintainer of the software *
  • The homepage of the software *
  • The license *
  • A brief description in one line *
  • A long description *

If the software needs to compile it should be compiled with rosmake. When the software is a ROS package or a stack it is sufficed to provide the information which are marked with a star (*) in the ROS manifest.xml or stack.xml.
